# 💰 Gestor de Gastos

Aplicação web para controle financeiro pessoal com autenticação, cadastro de receitas, registro de gastos, categorias personalizadas e gráficos interativos.

## 📌 Sobre o Projeto

O Gestor de Gastos é um sistema desenvolvido para ajudar usuários a controlarem melhor suas finanças mensais. A aplicação permite registrar receitas, adicionar despesas, organizar gastos por categoria e acompanhar o saldo final do mês.

O projeto conta com autenticação de usuários via Firebase, armazenamento em nuvem com Firestore e visualização gráfica dos dados financeiros.

---

## 🚀 Funcionalidades

* Login com e-mail e senha
* Cadastro de novos usuários
* Login com Google
* Registro de receita mensal
* Cadastro de gastos
* Criação de categorias personalizadas
* Filtro de gastos por mês
* Marcação de gastos como pendentes ou pagos
* Exclusão de gastos
* Resumo financeiro com receita, total gasto e saldo final
* Gráfico de receita x gasto
* Gráfico de gastos por categoria
* Tema claro e escuro
* Layout responsivo

## ⚙️ Como Funciona

1. O usuário faz login ou cria uma conta.
2. Os dados financeiros são carregados do Firebase.
3. O usuário informa a receita do mês.
4. O usuário cadastra gastos com descrição, valor, categoria e data.
5. O sistema calcula automaticamente o total gasto e o saldo final.
6. Os dados são exibidos em cards, listas e gráficos.
7. Todas as informações são salvas na nuvem por usuário.
